# CI/CD developers

CI/CD pipeline authoring. Cross-stack — invoke regardless of the project's primary language.

| Agent | Purpose | When to invoke |
|-------|---------|----------------|
| github-actions-developer | GitHub Actions workflows | Authoring or fixing workflows under `.github/workflows/`, composite actions, or reusable workflows. |
| gitlab-ci-developer | GitLab CI/CD pipelines | Authoring or fixing `.gitlab-ci.yml`, CI/CD components, or child pipelines. |
